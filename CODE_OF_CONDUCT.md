# Code of Conduct

## Our Standard

We want GroundedDeck to be a practical and technically serious open-source project.

Participants are expected to:

- discuss ideas in good faith
- criticize code and decisions, not people
- keep feedback concrete and actionable
- avoid harassment, discrimination, and abusive behavior

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate these standards.

## Reporting

Report conduct issues through a private security or maintainer contact channel once one is published in this repository.
