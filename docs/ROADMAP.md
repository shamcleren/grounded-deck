# Roadmap

## 中文说明

路线图描述 GroundedDeck 从脚手架到可演示产品的阶段性目标。里程碑应保持按能力递进，而不是先做 renderer 再回补规划能力。

## Milestone 1: Foundation

## 里程碑 1：基础阶段

- repository scaffold
- architecture docs
- `slide spec` schema
- deterministic self-acceptance harness

- 仓库脚手架
- 架构文档
- `slide spec` schema
- 确定性的 self-acceptance harness

## Milestone 2: Planning Pipeline

## 里程碑 2：规划流水线

- ingest source pack metadata
- normalize extracted source units
- compile deck goals and slide outlines
- emit schema-valid `slide spec` output

- 摄取 source pack 元数据
- 规范化提取出的 source units
- 编译 deck goals 和 slide outlines
- 输出 schema-valid 的 `slide spec`

## Milestone 3: Quality Gates

## 里程碑 3：质量门禁

- coverage grader
- grounding grader
- duplication and coherence grader
- deck review report generation

- coverage grader
- grounding grader
- duplication 和 coherence grader
- 生成 deck review 报告

## Milestone 4: Editable Rendering

## 里程碑 4：可编辑渲染

- native text rendering
- native tables and charts
- shape-based diagram templates
- Chinese-safe typography defaults

- 原生文本渲染
- 原生表格和图表
- 基于形状的图示模板
- 中文安全的排版默认值

## Milestone 5: Public Demo Quality

## 里程碑 5：公开演示质量

- sample datasets
- reproducible output examples
- benchmark rubric for release candidates

- 示例数据集
- 可复现实例输出
- 面向发布候选版本的 benchmark rubric
